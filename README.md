# ✈️ Plane Management System (Java)

## 📌 Overview  
This repository contains a **Plane Management System** built using the Java programming language.  
The system simulates basic airline seat booking and management operations in a structured console-based application.

---

## 🎯 Purpose  
- Apply Java programming concepts in a real-world simulation  
- Practice Object-Oriented Programming (OOP) principles  
- Build a structured airline seat management system  
- Improve problem-solving and logical thinking skills  
- Understand system flow and booking management logic  

---

## 🧠 Key Features  
- 🪑 Seat Booking System  
- ❌ Cancel Reservations  
- 🔍 View Available Seats  
- 👤 Passenger Management  
- 📊 Seat Status Tracking  
- 🧾 Booking Summary Output  

---

## 🛠️ Tech Stack  
- ☕ Java (Core Programming Language)  
- 💻 IDE: IntelliJ IDEA / Eclipse / NetBeans  
- 📁 Console-Based Application  
- 🧩 Object-Oriented Programming (OOP)  

---

## 🚀 Objective  
To develop a **plane seat management simulation system** that demonstrates strong Java fundamentals, structured programming, and real-world application logic.

---

## 📂 Project Highlights  
- Clean OOP-based architecture  
- Simple and efficient booking flow  
- Beginner-friendly airline system simulation  
- Strong focus on logic building and data handling
